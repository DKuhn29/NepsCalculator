# NepsCalculator
### A simple calculator written in C for a NepsAcademy project

## Features
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)

## Requirements
1. C Compiler (GCC recommended)
2. Operating system: Windows, macOS or Linux
